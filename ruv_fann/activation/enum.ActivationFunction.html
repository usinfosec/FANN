Activation functions available for neurons

These functions are based on the FANN libraryโs activation functions and include both common neural network activation functions and some specialized variants.

Threshold activation function: f(x) = 0 if x < 0, 1 if x >= 0 Note: Cannot be used during training due to zero derivative

Symmetric threshold: f(x) = -1 if x < 0, 1 if x >= 0 Note: Cannot be used during training due to zero derivative

Sigmoid

Sigmoid activation function: f(x) = 1 / (1 + exp(-2 * steepness * x)) Output range: (0, 1)

Symmetric sigmoid (tanh): f(x) = tanh(steepness * x) Output range: (-1, 1)

Gaussian

Gaussian activation: f(x) = exp(-x * steepness * x * steepness) Output range: (0, 1], peaks at x=0

ยง

GaussianSymmetric

Symmetric gaussian: f(x) = exp(-x * steepness * x * steepness) * 2 - 1 Output range: (-1, 1], peaks at x=0

ยง

Elliot

Elliott activation: f(x) = ((x * steepness) / 2) / (1 + |x * steepness|) + 0.5 Fast approximation to sigmoid, output range: (0, 1)

ยง

ElliotSymmetric

Symmetric Elliott: f(x) = (x * steepness) / (1 + |x * steepness|) Fast approximation to tanh, output range: (-1, 1)

ยง

LinearPiece

Bounded linear: f(x) = max(0, min(1, x * steepness)) Output range: [0, 1]

LinearPieceSymmetric
